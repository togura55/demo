BxtTool

Description:
 PDF XMPデータ領域からBaXterデータフォーマットを読みだして内容を表示するアプリケーションプログラム。またBaXterフォーム情報をCSVファイルにエクスポートする。

How To Use:
- bin\Bxttool.exe をダブルクリックして起動
- PDFファイルをDirボタンで指定して、Readボタンで読み込む。結果がリストボックスに表示される
- Exportボタンをクリックすると、bin\fields.csvファイルが生成される。

Environment:
- Windows 7 or later, .NET Framework 4.5.2 or later
